Exemplo de Tags de Mídia em HTML

Imagem

Esta é uma imagem de exemplo. O atributo alt fornece uma descrição para a imagem.

O que é: A tag <`img`> é usada para inserir imagens na página.

Atributos principais:

<`src`>: Especifica o caminho do arquivo da imagem.

<`alt`>: Fornece uma descrição alternativa para a imagem, importante para acessibilidade e SEO

Como utilizar:

<`img src="caminho/para/imagem.jpg" alt="Descrição da imagem"`>

Por que usar:

Para adicionar gráficos, fotos ou qualquer outro tipo de imagem à página. A descrição alternativa é crucial para usuários com deficiência visual e para SEO.

Áudio

Este é um exemplo de clipe de áudio. Use o controle para reproduzir o áudio.

O que é: A tag <`audio`> é usada para incorporar arquivos de áudio na página.

Atributos principais:

`controls`: Adiciona controles de reprodução (play, pause, volume) ao player de áudio.

src: Especifica o caminho do arquivo de áudio.

type: Informa o tipo MIME do arquivo de áudio.

Como utilizar:

<`audio controls`> <`source src="caminho/para/audio.mp3" type="audio/mpeg"`> Seu navegador não suporta o elemento de áudio. <`/audio`>

Por que usar:

Para incluir clipes de áudio, como músicas ou efeitos sonoros. O atributo controls permite que os usuários reproduzam, pausem e ajustem o volume do áudio.

Vídeo

Este é um exemplo de clipe de vídeo. Use o controle para reproduzir o vídeo.

O que é: A tag <`video`> é usada para incorporar vídeos na página.

Atributos principais:

width e height: Define as dimensões do vídeo.

controls: Adiciona controles de reprodução ao player de vídeo.

src: Especifica o caminho do arquivo de vídeo.

type: Informa o tipo MIME do arquivo de vídeo.

Como utilizar:

<`video width="640" height="360" controls`> <`source src="caminho/para/video.mp4" type="video/mp4"`> Seu navegador não suporta o elemento de vídeo. <`/video`>

Por que usar:

Para incorporar clipes de vídeo, como tutoriais, trailers ou outros conteúdos visuais. O atributo controls permite que os usuários interajam com o vídeo.

Imagem com Legenda

Usamos a tag <figure> para agrupar a imagem e a <figcaption> para adicionar uma legenda.

O que são: A tag <`figure> é usada para agrupar mídia (como imagens, vídeos) e sua descrição. A tag <`figcaption> fornece uma legenda para o conteúdo da <`figure>.

Como utilizar:

<`figure> <`img src="caminho/para/imagem.jpg" alt="Descrição da imagem"> <`figcaption>Esta é a legenda da imagem. <`/figure>

Por que usar:

Para fornecer contexto e descrição adicional para elementos de mídia, melhorando a acessibilidade e a compreensão do conteúdo.

Iframe

Este é um exemplo de um iframe incorporando uma página da web externa.

O que é: A tag <`iframe> permite incorporar outra página da web dentro da sua página.

Atributos principais:

src: Especifica a URL da página a ser incorporada.

width e height: Define as dimensões do iframe.

frameborder: Remove a borda ao redor do iframe (0 remove a borda).

Como utilizar:

<`iframe src="https://www.example.com" width="600" height="400" frameborder="0"> Seu navegador não suporta iframes. <`/iframe>

Por que usar:

Para incluir conteúdo de outras páginas da web, como mapas, vídeos do YouTube ou widgets de redes sociais.
